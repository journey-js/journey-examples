Basic Component

In this demo, we show how to create a basic component with Ractive.

Below is the component in action!

{{feedback}}

Below we create the component: 

import compTemplate from './compTemplate.html';

function createComponent() {

			 var comp = ractive.extend({
                template: compTemplate,
                activate: function() {
                    this.set( "hidden", false );
					this.set( "feedback", "Component clicked!" );
                }                
            });
};

To use the component in our view we declare it with the components property under the name we want to use in our markup. In this case we call our component simple.

function createView() {

    var comp = createComponent():

    var view = new template({
        components: {
            simple: comp
        }
    });
};

Below we show how the component is used by the view:

We can map data from the view to the component through attributes when declaring the component.

In the snippet above we map 3 values, message, hidden and feedback.

The message is passed from the view to the component and displayed on the component through the \{{message}} mustache.

The hidden variable is mapped to the component so when the component changes hidden to true, the view' hidden variable is updated as well, meaning the class-hidden directive becomes false and the feedback box is displayed.

The feedback variables that is mapped to the component allows the component to set the feedback message that the view will display.
