Method Calls

One of the first things we want to do in SPA is calling a Javascript methods from html templates. This is easily achieved with Ractive.

Click here, and view the output in your browser console.

Let's quickly look at the template where we declare the Ractive on-click directive.

methods.html:

In the link above we declare the on-click directive to call the method test(). The @. prefix tells Ractive to invoke the method on the Ractive instance. The reason for the @. prefix is because the template can invoke methods in different locations such as @global which is generally the window object.

Beloww is the view with the test() method.

methods.js: 
...
enter: function ( route, prevRoute, options ) {
		route.view = createView( options.target );
	}

	function createView( target ) {

		var view = new Ractive( {
			el: target,
			template: template,

			test: function ( ) {
			console.log( "test() invoked!" );
			return false; //returning false cancels the original event, effectively calling event.preventDefault() and event.stopPropagation()
		},
	}

We can see the test() method defined on the Ractive instance.
