Navigation Demos

Journey supports two ways to navigate between routes:

  • programmatically with journey.goto( '/product' )
  • declaratively through links <a href="/person">

Navigate Programatically

import journey from 'lib/journey';
import product from 'app/views/product/product.js';

// We add a route for product that accepts an 'id' parameter
journey.add('/product/:id', product);

var id = 2; // We goto product passing the 'id'
journey.goto( '/product/' + id);

Navigate Programatically with data

Journey does not support passing data through routes. However with ES6 Modules you can easily store data in a module, say session.js, and retrieve the data from the session in the target route. 
import journey from 'lib/journey';
import productSearch from 'app/views/product/productSearch.js';
import session from 'app/util/session.js';

var products = getProductsFromServer();

// Store products in session
session.products products;

journey.goto( '/productsearch');

// In productsearch.js
import session from 'app/util/session.js';

// Retrieve products from session
var products = session.products;
