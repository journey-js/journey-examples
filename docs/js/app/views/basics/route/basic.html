Basic Route

This example shows how to implement a basic Journey route.

Routes in Journey are plain Javascript objects that must implement the enter method. When we navigate to a route the enter is called which is where we will create our views. We can also implement a leave method which is called when we navigate to a new route. leave is ideal for remove the view from the dom.

In these examples we use ES6 Modules and split our views and templates into separate modules. We use a Node and Rollup to develop and distribute these examples.

Let's look at our first Journey route. below, basic.js: 

import Ractive from "lib/Ractive.js";
import template from "./basic.html";

var basic = {

	enter: function ( route, prevRoute, options ) {

		// We create our view (using Ractive) and assign it to a property on the route object
		route.view = new Ractive( {

			// options.target is the default target configured in start.js
			el: options.target,

			// we use the html template imported above
			template: template
		} );
	},

	leave: function ( route, nextRoute, options ) {
		// When we leave the route we teardown the view.
		// teardown() is a Ractive method to remove and destroy the view
		return route.view.teardown();
	}
};

export default basic;

In line 1 and 2, we import Ractive and our view template.

Line 4 is our route object basic.

In line 6 we implement the enter method and line 9 we create our view with Ractive. line 19 implements the leave method where the view is destroyed.
