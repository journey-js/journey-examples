Basic Partial

Partials are template snippets that can be embedded into other templates. Click here for details.

Below is an example partial

{{>hello}}

Here is the Javascript code:

 import helloPartial from './partial.html';

let ractive = new Ractive({
	el: '#main',
	partials: {hello: helloPartial}
});

In line 5 we add our helloPartial to the property, hello.

Next is the template which references the partial:

 \{{>hello}}

In line 1 we use > character inside a mustache to reference the hello partial.

And finally the actual partial:

<h3 class="partial">
    Hi, I am a partial!
</h3>
